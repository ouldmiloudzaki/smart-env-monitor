About this project

Smart Environmental Monitor is an end-to-end AI + IoT demo. A virtual sensor continuously reports temperature, humidity, and air quality; the data is streamed live to an AI model that learns what "normal" looks like and flags abnormal readings the instant they happen — the same pipeline behind predictive maintenance, smart agriculture, and air-quality systems.

01 · IoT

Virtual Sensor

Generates realistic readings every 2 seconds, with occasional spikes to simulate real events.

02 · Backend

FastAPI + WebSocket

Receives each reading and streams it to the dashboard over a live connection — no page refresh.

03 · AI

Isolation Forest

An ML model trained on normal data scores every reading and detects anomalies with a confidence level.

04 · UI

Live Dashboard

Shows the values, a real-time chart, and an event feed — anomalies are highlighted in red.
